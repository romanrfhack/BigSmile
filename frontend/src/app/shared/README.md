# Shared

Reusable UI components, directives, pipes, and utilities that are used across multiple features.

- **UI components**: Buttons, modals, form controls, tables, etc.
- **Directives**: Custom Angular directives (tooltips, permissions, etc.)
- **Pipes**: Formatting pipes (currency, date, etc.)
- **Shared services**: Generic services (logging, local storage, etc.)

Shared modules may depend on Core but not on Shell or Feature modules.
